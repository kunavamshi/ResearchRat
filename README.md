## âš™ï¸ Setup & Installation

### 1. Prerequisites
- Node.js installed
- MySQL Server installed and running

### 2. Backend Setup
1. Navigate to the backend folder:
   ```sh
   cd backend
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Configure environment variables:
   - Open `.env` and update `DB_USER` and `DB_PASSWORD` with your MySQL credentials.
   ```env
   PORT=5000
   DB_HOST=localhost
   DB_USER=root
   DB_PASSWORD=your_password
   DB_NAME=expense_tracker
   ```
4. Setup Database:
   - You need to run the `schema.sql` script in your MySQL Workbench or CLI.
   ```sh
   mysql -u root -p < schema.sql
   ```
   *Note: This will create the `expense_tracker` database and `expenses` table.*

5. Start the server:
   ```sh
   npm run dev
   ```
   Server will run on `http://localhost:5000`.

### 3. Frontend Setup
1. Navigate to the `frontend` folder.
2. Open `index.html` in your browser.
   - You can use a live server extension or simply double-click the file.
